This is a list of tools that run locally via your browser which are helpful for creating material on my.wbs

* [Petrock](petrock.html) - A transcript formatter for Otter/Text
* [Varnisher](varnisher.html) - Cleans up source code for my.wbs
* [Skippy](skippy.html) - Create a jump link box by pasting in the page's HTML
* [Southpaw](southpaw.html) - For creating clean box and activity templates
* [Tafelberg](tafelberg.html) - Helps to make tables for my.wbs and reformat existing ones  *- Note that this one is very WIP*
